# Primary Category Plugin
**Contributors:** Tommy Ferry
**Requires at least:** 5.1
**Tested up to:** 5.3.2
**Stable tag:** 1.0.0

Adds the ability to set a primary category for posts.

## Description

Adds the ability to set a primary category for posts in the Gutenberg editor.

You can also add the ability to add a primary term to custom (and other core) taxonomies, by using the following filters:
- `pcp_primary_term_taxonomies`
- `pcp_primary_term_taxonomies_args`

## Installation

### Manual Installation

1. Upload the entire `/primary-category-plugin` directory to the `/wp-content/plugins/` directory.
2. Activate Primary Category Plugin through the 'Plugins' menu in WordPress.
